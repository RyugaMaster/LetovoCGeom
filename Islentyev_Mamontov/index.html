Алгоритм Джарвиса

Алгоритм Джарвиса определяет последовательность элементов множества, образующих выпуклую оболочку для этого множества. Метод можно представить как обтягивание верёвкой множества вбитых в доску гвоздей.

Выпуклой оболочкой множества точек называется выпуклый многоугольник, построенный вершинами на этих точках, содержащий в себе все точки этого множества.

Представьте себе доску, в которую вбито — но не по самую шляпку — много гвоздей. Возьмите верёвку, свяжите на ней скользящую петлю и набросьте её на доску, а потом затяните. Верёвка окружает все гвозди, но касается она только некоторых, самых внешних. Те гвозди, которых она касается, составляют выпуклую оболочку для всей группы гвоздей.

Допустим, у нас есть множество точек. Нам нужно построить выпуклую оболочку. Алгоритм Джарвиса позволяет нам найти из множества точек только те, которые входят в выпуклую оболочку.

Суть алгоритма Джарвиса заключается в нахождении следующей точки выпуклой оболочки за линейное время, если известна предыдущая точка F(x’; y’). Чтобы найти следующую точку A, нужно найти точку с минимальным углом между прямой y = y’ и лучом FA. Это можно сделать пробегом по всем точкам, сравнивая векторные произведения. Когда найденной точкой окажется самая первая - алгоритм останавливается. Выпуклая оболочка построена.

Количество точек в выпуклой оболочке не превосходит n, изначальный размер множества. Так как для каждой точки мы запускаем цикл по всем оставшимся, то сложность алгоритма составит:

n + (n – 1) + (n – 2) + … + 1 = (n * (n + 1)) / 2 = 0.5n2 + 0.5n ∈ O(n2)
